# Django Storefront

A simple ecommerce backend built with Django and Django REST Framework.

## Features

- Product catalog
- Shopping cart
- Checkout system
- REST API
- Bootstrap frontend

## Installation

Clone the repository

git clone https://github.com/vince09-G/storefront.git

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Run server

python manage.py runserver
